# Авторизация — Тест-кейсы

## TC-001 Успешная авторизация

### Шаги:
1. Открыть страницу входа
2. Ввести валидный логин
3. Ввести валидный пароль
4. Нажать "Войти"

### Ожидаемый результат:
Пользователь успешно авторизован.

---

## TC-002 Авторизация с пустым паролем

### Шаги:
1. Ввести логин
2. Оставить пароль пустым
3. Нажать "Войти"

### Ожидаемый результат:
Появляется сообщение об ошибке.
